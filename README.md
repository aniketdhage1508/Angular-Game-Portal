
# **Angular Game Portal**

Welcome to the **Angular Game Portal** project! This is a web-based game information portal built using **Angular**, where users can explore various games and their details.

## **Features**

- **Game List**: Browse a list of games with their details, such as name, description, release date, and genre.
- **Game Search**: Search for games by name, genre, or release year.
- **Game Details**: View detailed information about each game, including screenshots, trailers, and reviews.
- **Responsive Design**: Fully responsive layout for a smooth experience across all devices.

## **Technologies Used**

- **Frontend**: Angular
- **Backend**: None (Client-side only)
- **API**: Fetches game data from a third-party API (e.g., RAWG, IGDB)

## **Setup Instructions**

1. Clone the repository:
   ```bash
   git clone https://github.com/aniketdhage1508/Angular-Game-Portal.git
   ```

2. Navigate into the project folder:
   ```bash
   cd Angular-Game-Portal
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   ng serve
   ```

5. Open the project in your browser:
   - Go to `http://localhost:4200`
